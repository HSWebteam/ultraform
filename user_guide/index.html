Welcome to Ultraform

Ultraform is an form generation library for the Codeigniter framework written in PHP. Its goal is to enable you to forms much faster than you could if you were writing code from scratch. It includes full client side validation through javascript, for more features check below.


This is a alpha release. Work continues on further alpha releases on github.

We are trying to keep the interface largely the same, but things are bound to change. Use for production environments at your own risk.
