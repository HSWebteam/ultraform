Credits

Ultraform has been written, and is being maintained, by H&S Webteam. Ultraform was written to be a solution for form generation with proper server and client validation build in.

Ultraform is currently working on several alphas to get to a beta release and in the future release candidates.

The team was inspired by the Form Generation Library by Frank Michel.
